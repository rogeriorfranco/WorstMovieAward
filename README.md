# Movie Awards API

## Como rodar a aplicação
1. Clone o repositório:

```bash
git clone https://github.com/rogeriorfranco/WorstMovieAward.git

cd Worst.Movie.Award.UserInterface

2 Instale as dependências:
dotnet restore
 
3 Rode a aplicação:
dotnet run

4 Acesse o endpoint:
GET http://localhost:8080/awards/api
